Ansible plugin for RabbitMQ
===========================

A set of ansible plugins to interact with RabbitMQ

Modules
=======
 - rabbitmq_user - Add/remove users and permissions in RabbitMQ
 - rabbitmq_plugin - Enable/disable plugins in RabbitMQ

===================
